# MathBoard — Product Roadmap & Feature Specification

## Цель проекта

MathBoard — приложение для интерактивных досок и сенсорных экранов, предназначенное для проведения уроков математики.

Основная цель проекта — минимизировать количество действий, не связанных непосредственно с объяснением материала. Учитель должен иметь возможность проводить весь урок стилусом, не перемещаясь к панелям инструментов и не переключаясь между несколькими приложениями.

MathBoard не является аналогом графического редактора. Основной сценарий использования — проведение уроков математики на интерактивной доске.

# Release 1.0 — Core Teaching Board

Версия считается завершённой, если позволяет провести полноценный урок без использования стороннего ПО.

## Infinite Canvas

### Описание

Бесконечный холст без фиксированных границ.

### Требования

* Холст не имеет визуальных границ.
* Поддерживается панорамирование.
* Поддерживается масштабирование.
* Положение камеры сохраняется внутри документа.

---

## Stroke System

### Описание

Основной механизм рукописного ввода.

### Требования

* Рисование стилусом.
* Рисование мышью.
* Настраиваемая толщина линии.
* Настраиваемый цвет линии.
* Сглаживание линий.
* Хранение линий как отдельных объектов Stroke.

### Ограничения

Ластик не должен рисовать белым цветом.

Удаление выполняется через удаление объектов Stroke.

---

## Eraser Tool

### Описание

Инструмент удаления рукописного ввода.

### Требования

* Удаляет Stroke полностью.
* Поддерживает удаление нескольких Stroke подряд.
* Работает в реальном времени.

---

## Quick Access Menu

### Описание

Основной способ выбора инструментов.

### Требования

* Может быть вызван без перемещения к краям экрана.
* Открывается рядом со стилусом.
* Позволяет выбрать:

  * кисть;
  * ластик;
  * цвет;
  * толщину линии;
  * выделение.

### Цель

Любой инструмент должен выбираться максимум за одно действие.

---

## Lesson Library

### Описание

Система организации материалов.

### Структура

Class
└─ Subject
└─ Topic
└─ Lesson

### Требования

* Создание классов.
* Создание тем.
* Создание уроков.
* Переименование элементов.
* Удаление элементов.
* Автосохранение.

---

## Persistence

### Описание

Сохранение состояния уроков.

### Требования

* Сохранение документов на диск.
* Загрузка документов.
* Автосохранение.
* Восстановление после аварийного завершения.

---

## Export

### Описание

Экспорт результатов урока.

### Требования

* PNG.
* PDF.

# Release 1.5 — Teaching Tools

Функции, повышающие удобство проведения уроков.

---

## Shape Tool

### Описание

Создание базовых фигур.

### Поддерживаемые объекты

* линия;
* стрелка;
* прямоугольник;
* окружность;
* треугольник.

### Требования

После создания фигура остаётся редактируемым объектом.

---

## Selection Tool

### Описание

Работа с существующими объектами.

### Возможности

* выделение;
* перемещение;
* копирование;
* удаление.

---

## Grid System

### Описание

Отображение математических сеток.

### Режимы

* без сетки;
* квадратная сетка;
* координатная плоскость.

### Требования

Сетка не должна влиять на содержимое документа.

---

## Alignment Helpers

### Описание

Помощь при построении чертежей.

### Возможности

* привязка к горизонтали;
* привязка к вертикали;
* отображение направляющих.

---

# Release 2.0 — Mathematics Features

Версия, превращающая MathBoard из доски в математический инструмент.

---

## Function Graphs

### Описание

Построение графиков функций.

### Требования

* Ввод аналитического выражения.
* Автоматическое построение графика.
* Масштабирование.
* Настройка диапазона отображения.

### Примеры

* y = x²
* y = sin(x)
* y = |x|

---

## Coordinate Plane Object

### Описание

Полноценная координатная система как отдельный объект.

### Возможности

* оси координат;
* деления;
* подписи;
* масштабирование.

---

## Geometry Objects

### Описание

Редактируемые математические объекты.

### Типы

* точка;
* отрезок;
* луч;
* прямая;
* окружность;
* угол.

### Требования

Объекты должны оставаться математически корректными после редактирования.
